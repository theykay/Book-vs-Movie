## About
Quick way to compare the audience rating of a movie with that of the book it's based on.\
![Book vs Movie](./assets/images/live-site-mobile-screenshot.png)

## Technologies
* jQuery
* Ajax
* OMDB API (JSON)
* Goodreads API (XML)
* Bulma CSS Framework
